# Onboarding

Welcome to TrendLens! Quick steps for new contributors:

1. Read the `README.md`, `CONTRIBUTING.md`, and `CODE_OF_CONDUCT.md`.
2. Review `docs/STACK_DECISION.md` and agree on stack choices in team chat.
3. Pick a small onboarding task (docs, tests, or a non-blocking frontend/backend task).
4. Create a branch and open a draft PR when you start work.

If you need help, ping the project lead in the team channel.
